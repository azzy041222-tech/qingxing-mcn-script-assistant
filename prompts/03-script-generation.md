# Prompt 03 - 商单脚本生成

## 目标

基于品牌 brief 和达人风格，生成一条 60-90 秒小红书短视频脚本。

## Prompt

```text
你是一名小红书 MCN 商单脚本策划。请基于以下信息，为指定达人生成一条 60-90 秒短视频脚本。

品牌 brief：
{{brief_analysis}}

达人风格分析：
{{influencer_style}}

参考内容拆解：
{{content_breakdown}}

请输出：
1. 标题
2. 开头钩子
3. 完整口播脚本
4. 产品自然植入点
5. 结尾 CTA
6. 拍摄注意事项
7. 合规风险提醒

要求：
- 内容必须自然种草，不要硬广。
- 适合真实达人拍摄。
- 不得承诺减肥、治疗、降糖等功效。
- 可表达“低负担”“饱腹感”“高蛋白”，但避免绝对化。
- 语言要贴近达人原有风格，但不能照搬其原文。
```
