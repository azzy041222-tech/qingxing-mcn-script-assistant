# Prompt 02 - 达人风格分析

## 目标

根据小红书博主主页和代表内容，拆解其内容风格，用于后续脚本生成。

## Prompt

```text
你是一名 MCN 达人内容分析师。请根据以下博主信息，分析该博主是否适合品牌商单，并拆解其内容风格。

品牌 brief：
{{brief_analysis}}

博主信息：
{{influencer_profile}}

代表内容：
{{representative_posts}}

请输出：
1. 博主人设
2. 主要内容方向
3. 粉丝画像推测
4. 常用开头钩子
5. 常见视频结构
6. 镜头与场景特点
7. 语言风格
8. 与品牌的匹配点
9. 不适合或需规避的风险
10. 可参考但不能照搬的表达方式

要求：
- 不要虚构未提供的数据。
- 如果信息不足，请标注“需人工补充”。
- 不要建议直接照搬原博主文案。
```
