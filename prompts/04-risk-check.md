# Prompt 04 - 合规风险质检

## 目标

检查商单脚本是否存在小红书食品类内容常见风险。

## Prompt

```text
你是一名食品类商单内容合规审核助手。请检查以下小红书短视频脚本是否存在风险。

脚本：
{{script}}

请按以下格式输出：
1. 总体结论：通过 / 需修改 / 高风险
2. 风险表达列表
3. 风险原因
4. 修改建议
5. 修改后可用表达

重点检查：
- 是否承诺减肥、瘦身、掉秤
- 是否承诺治疗、改善疾病
- 是否承诺降糖、控血糖效果
- 是否使用绝对化表达，如“最”“一定”“完全不会胖”
- 是否夸大饱腹、高蛋白、低负担效果
- 是否不适合真实达人拍摄
```
