# 轻醒 x 凌二七脚本合规风险质检

## 总体结论

通过，建议录制前复核产品包装营养数据。

当前脚本已避免减肥、治疗、降糖、替代正餐等食品类内容高风险表达，主要使用“0 蔗糖”“高蛋白”“希腊酸奶”“低负担”“早餐搭配”“运动后加餐”等相对安全的产品属性与食用场景表达。

## 已规避表达

- 减肥、瘦身、燃脂
- 降糖、控血糖、适合糖尿病人
- 治疗、改善疾病
- 代餐神器、替代正餐
- 吃了不胖、随便吃无负担
- 保证撑到中午、一定掉秤等绝对化承诺

## 可使用表达

- 0 蔗糖
- 高蛋白
- 希腊酸奶
- 低负担
- 饱腹感
- 早餐搭配
- 运动后加餐
- 下午茶搭子
- 对我来说比较顶饱

## 需谨慎表达

| 表达 | 风险 | 建议 |
| --- | --- | --- |
| 控糖友好 | 可能被理解为血糖功效 | 优先使用“0 蔗糖配方” |
| 代餐 | 容易触发替代正餐风险 | 改为“早餐搭配”“加餐选择” |
| 减脂餐 | 可能被理解为减肥承诺 | 改为“轻食搭配”“低负担搭配” |
| 撑到中午 | 个人效果承诺偏强 | 改为“对我来说比较顶饱” |
